# Diceware wordlist attribution

These Diceware wordlists were imported from the MIT-licensed
[SpamOK.PasswordGenerator](https://github.com/SpamOK/SpamOK.PasswordGenerator)
project. Each list contains exactly 7776 words (6^5), one word per line, in dice
order.

SpamOK.PasswordGenerator is licensed under the MIT license. The individual wordlists 
were collected by that project from the following upstream sources:

| File          | Language   | Upstream source |
|---------------|------------|-----------------|
| `en.diceware` | English    | theworld.com (Arnold Reinhold's original Diceware list) |
| `nl.diceware` | Dutch      | https://mko.re/diceware/ |
| `de.diceware` | German     | https://github.com/dys2p/wordlists-de |
| `fr.diceware` | French     | https://github.com/ArthurPons/diceware-fr-alt |
| `es.diceware` | Spanish    | https://github.com/mir123/dadoware-bonito-es |
| `it.diceware` | Italian    | https://www.taringamberini.com/ |
| `ro.diceware` | Romanian   | Community contribution |

Refer to each upstream source for its specific license terms. If you redistribute
these lists, preserve this attribution.

Files marked as "Community contribution" were contributed by members of the community. If you are the original author or know the original upstream source, please let us know so we can provide proper attribution.
