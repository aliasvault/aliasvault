# Android app
You can install the AliasVault Android app from several sources. Pick whichever you prefer:

## Download from Google Play
1. Go to the [Google Play Store](https://play.google.com/store/apps/details?id=net.aliasvault.app)
2. Click on the "Install" button
3. The app will be installed. Open the app and follow the instructions.

## Download from F-Droid
[F-Droid](https://f-droid.org) is an open-source app store. The AliasVault build there is compiled from source by F-Droid and contains no proprietary dependencies.

1. Open the [AliasVault page on F-Droid](https://f-droid.org/packages/net.aliasvault.app/) (the [F-Droid client](https://f-droid.org/#what-is-fdroid) is recommended for automatic updates)
2. Tap "Install"
3. Open the app and follow the instructions.

## Build from Source
If you wish to install the mobile app from source instead, see the [build-from-source](build-from-source.md) documentation. This will allow you to make changes to the mobile app and/or to use a specific version of the mobile app.
